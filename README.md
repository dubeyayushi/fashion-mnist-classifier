# Fashion MNIST Classifier

A web app to classify clothing images into 10 categories using a CNN model trained on Fashion MNIST.

- **Live Demo:** [https://fashion-mnist-classifier-6w71.onrender.com/](https://fashion-mnist-classifier-6w71.onrender.com/)
- **Tech Stack:** FastAPI, TensorFlow/Keras, Jinja2, HTML/CSS

## Usage
1. Go to the live demo link above.
2. Upload a clothing image (28x28 grayscale or any clothing photo).
3. Get instant AI predictions and class probabilities.
